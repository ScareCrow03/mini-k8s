| 请求类型 | URI               | 描述                           |
| -------- | ----------------- | ------------------------------ |
| POST     | /createFromFile   | 创建某一个实例                 |
| POST     | /applyFromFile    | 修改某个实例                   |
| POST     | /deleteFromFile   | 删除某个实例                   |
| POST     | /getByType        | 获取某一个类型的实例的粗略信息 |
| POST     | /getByTypeAndName | 获取特定名字的实例的粗略信息   |
| POST     | /describeByType   | 详细描述某一类型实例的信息     |
